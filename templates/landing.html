Everything TwinGuard does

A complete passive WiFi threat detection suite built for security-conscious users who want local control.

Passive WiFi Scanning
Scans nearby networks without connecting. Extracts SSID, BSSID, signal strength, channel, and auth type at Layer 2.
Evil Twin Detection
Identifies duplicate SSIDs with MAC prefix mismatches, vendor OUI conflicts, and security downgrades.
Enterprise Recognition
Intelligently identifies dual-band routers and campus multi-AP deployments, preventing false positives.
Real-Time Alerts
Centered modal popup fires immediately when a suspicious network is detected with visual severity indication.
Live Dashboard
Color-coded risk levels in a responsive table. Dynamic JavaScript updates without any page reload.
Lightweight Stack
Python + HTML/CSS/JS . No heavy dependencies, minimal resource usage, zero cloud infrastructure.
Detection Engine

Rule-Based Threat Classification

Four risk levels assigned to every scanned network using multi-factor analysis.

Safe — No threats detectedSafe
Dual Band / Enterprise networkEnterprise
Security downgrade detectedHigh Risk
Duplicate SSID + MAC mismatchEvil Twin
1
Duplicate SSID Detection
Flags any two networks broadcasting the same name with different MAC addresses as suspicious.
2
MAC Vendor Prefix Check
Compares OUI (first 3 bytes of MAC) against a vendor database. Mismatches on the same SSID indicate spoofing.
3
Security Mismatch Analysis
Detects when one SSID instance uses WPA2/WPA3 while another is Open or WEP — a classic evil twin lure.
4
Enterprise False Positive Guard
Recognises dual-band and multi-AP deployments to avoid flagging legitimate enterprise infrastructure.
Process

Four steps, instant results

TwinGuard completes a full scan and analysis in under a second on most machines.

1
Click Scan
Trigger a request to the local Flask server running on port 5000 from your browser.
2
WLAN Query
Backend executes netsh wlan show networks, parsing all visible SSIDs, BSSIDs, channels, and security types.
3
Rule Engine
Each network is classified using 4 detection rules. Results returned as structured JSON to the frontend.
4
Alert & Review
Dashboard renders color-coded results. Threats trigger an immediate modal so you can act before connecting.
